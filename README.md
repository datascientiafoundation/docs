# Datascientia documentation

This document is written in markdown and compiled using [Quarto](https://quarto.org).
It can be converted to multiple formats (e.g., HTML, Latex, docx,...)

To preview it as a website

```bash
quarto preview
```

To render it as HTML website

```bash
quarto render
```

To build the website and make the changes public, just push the changes to this repository.
